# Meridian

Meridian is a web application designed to help users explore career options, access guidance, and find resources for professional development.

## Features
- Browse career paths and industries
- Access expert advice and articles
- Personalized recommendations
- Resource library for skill development

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/careerguide.git
   ```
2. Navigate to the project directory:
   ```bash
   cd careerguide
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
Start the development server:
```bash
npm start
```
Open your browser and go to `http://localhost:3000`.

## Technologies Used
- Sass for styling
- JavaScript/React (if applicable)
